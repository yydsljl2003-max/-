# 🐍 贪吃蛇 3D

一个使用 Three.js 构建的 3D 贪吃蛇游戏。

## 特性

- **Three.js 3D 渲染**：实时阴影、雾效、粒子系统、HDR 色调映射
- **三种视角**：等距视角 / 第三人称 / 第一人称（按 V 切换）
- **相对方向控制**：第一/第三人称下，WASD 始终相对于蛇的朝向
- **视觉特效**：吃到食物粒子爆发、屏幕震动、头部拖尾、食物环绕粒子
- **音效**：Web Audio API 合成音效（吃食物、死亡、移动）
- **金色奖励食物**：15% 概率出现，价值 30 分
- **渐进难度**：速度随分数逐渐加快，右下角速度条可视化
- **分数飘字**：+10/+30 浮动文字特效
- **移动端支持**：触屏按钮 + 滑动操作

## 如何游玩

直接在浏览器中打开 `index.html`，或使用任意静态服务器。

| 按键 | 功能 |
|------|------|
| 方向键 / WASD | 移动 |
| 空格 / ESC | 暂停/继续 |
| V | 切换视角 |

## 文件结构

| 文件 | 作用 |
|------|------|
| `index.html` | 入口页面 |
| `style.css` | 样式与动画 |
| `scene.js` | Three.js 场景、光照、摄像机、网格、粒子 |
| `input.js` | 键盘、触屏输入绑定 |
| `game.js` | 游戏逻辑、状态机 |
| `audio.js` | Web Audio 音效合成 |

## 技术栈

- Three.js 0.137
- Web Audio API
- 原生 JavaScript
